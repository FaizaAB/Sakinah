# Sakinah

Sakinah is a faith-aligned mental health platform for Muslims. It connects users with Muslim providers, scholar-informed guidance, community support, educational resources, and AI-assisted matching through Khalil.

## Features
- Provider matching
- Scholar-informed Islamic guidance
- Community support
- Educational content
- Financial aid pathways
- AI intake and matching
